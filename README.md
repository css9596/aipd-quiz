## 🎯 이렇게 공부하세요

처음 오셨다면 아래 순서를 추천합니다.

### 1️⃣ 먼저 실력 확인 — `시험 모드`
**20문항**으로 가볍게 시작해 보세요. 타이머가 돌아가고, 다 풀면 한 번에 채점됩니다.
지금 내 점수가 몇 점인지, 어디가 약한지 바로 드러납니다.

### 2️⃣ 약한 단원 집중 공략 — `성적 통계` → `주제별 학습`
**성적 통계**에서 장별 정답률을 확인하세요. 가장 약한 단원은 **빨간 테두리**로 표시됩니다.
그 장을 눌러 바로 집중 학습할 수 있어요.

### 3️⃣ 틀린 문제만 반복 — `오답노트`
틀린 문제는 **자동으로** 오답노트에 쌓입니다. 따로 정리할 필요가 없어요.
다시 풀어서 맞히면 목록에서 사라집니다. **오답노트가 비워질 때까지 반복**하는 게 가장 효율적인 공부법입니다.

### 4️⃣ 실전 감각 — `정답 위치 섞기` ON
같은 문제를 반복하면 "정답은 3번"처럼 **위치를 외워버리는** 함정에 빠집니다.
홈 화면에서 **정답 위치 섞기**를 켜두면 매번 보기 순서가 바뀌어, 진짜 실력으로 풀게 됩니다.

### 5️⃣ 마무리 점검 — `시험 모드` 전체
시험 직전에는 **전체 167문항**으로 최종 점검하세요.

| 메뉴 | 이럴 때 쓰세요 |
|---|---|
| **학습 모드** | 처음부터 차근차근. 답을 고르면 **즉시 채점 + 해설**이 나옵니다 |
| **시험 모드 (CBT)** | 실전처럼. 10 / 20 / 40 / 전체 중 선택 → 랜덤 출제 → 타이머 → 한 번에 채점 |

## ❓ 자주 묻는 질문

**Q. 회원가입이나 로그인이 필요한가요?**
아니요. 링크만 열면 바로 사용할 수 있습니다. 요금도 없습니다.

**Q. 공부 기록은 어디에 저장되나요?**
**내 휴대폰(브라우저) 안에만** 저장됩니다. 서버로 전송되지 않고, 다른 사람이 볼 수 없습니다.

**Q. 다른 기기에서도 이어서 볼 수 있나요?**
아니요. 기록은 기기별로 따로 저장됩니다. 폰에서 푼 기록은 PC에 나타나지 않습니다.

**Q. 기록을 지우고 처음부터 다시 하려면?**
홈 화면 맨 아래 **"기록 초기화"** 를 누르세요. (되돌릴 수 없으니 주의)

**Q. 문제를 풀다가 앱을 닫으면 처음부터 다시 해야 하나요?**
아니요. 다시 열면 홈에 **"이어서 풀기"** 버튼이 나타나 그 자리에서 이어집니다.

**Q. 정답이 이상한 것 같아요.**
정답·해설은 교재 원본을 그대로 따르고 있습니다. 오류를 발견하시면 알려주세요.
